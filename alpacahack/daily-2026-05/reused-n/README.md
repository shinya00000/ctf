# reused-n writeup

## 1. 概要

本問題は同一の平文 $m$ と同一のモジュラス $n$ に対して異なる公開指数 $e_1$ と $e_2$ を用いて暗号化された2つの暗号文 $c_1$ と $c_2$ から元の平文を復元する問題です．

## 2. 問題の分析

### ファイル構成

* chall.py
* chall.txt

### ソースコードの確認

ソースコードを確認するとflagがそのまま整数 $m$ に変換されています．
その後 $n = p \times q$ という1つのモジュラスに対して $e_1 = 1234$ と $e_2 = 5678$ という2つの異なる指数で暗号化処理が行われています．
フラグの長さが100バイト未満であるというアサーションも存在します．

## 3. 脆弱性と攻略方針

### 脆弱性

同一のモジュラス$n$を用いて同じ平文$m$を異なる指数で暗号化している点は共通モジュラス攻撃の対象となります．
通常の共通モジュラス攻撃では $\gcd(e_1, e_2) = 1$ であることが前提ですが本問題では $\gcd(1234, 5678) = 2$ となっています．

### 攻略方針：

拡張ユークリッドの互除法を用いると $a \cdot e_1 + b \cdot e_2 = \gcd(e_1, e_2) = 2$ を満たす整数 $a$ と $b$ を求めることができます．
これを利用することで $c_1^a \cdot c_2^b \equiv m^{a \cdot e_1 + b \cdot e_2} \equiv m^2 \pmod n$ を計算できます．
ここでフラグの長さは最大でも100バイトであるため $m < 2^{800}$ となります．
$n$ は2048ビットであるため $m^2 < n$ が成立します．
したがって剰余をとる前の $m^2$ と等しくなるため標準ライブラリの`math.isqrt`を用いて単純に平方根を計算することで $m$ を求めることが可能です．

## 4. エクスプロイトコード

```python
import math
from Crypto.Util.number import long_to_bytes

# 拡張ユークリッドの互除法
def egcd(a, b):
    if a == 0:
        return b, 0, 1
    else:
        g, y, x = egcd(b % a, a)
        return g, x - (b // a) * y, y

n, e1, c1, e2, c2 = (省略)
# 拡張ユークリッドの互除法で a*e1 + b*e2 = gcd(e1, e2) を満たす a, b を求める
g, a, b = egcd(e1, e2)

# c1^a * c2^b mod n を計算
# a または b が負の場合は pow(c, -1, n) で法 n における逆元を求める
if a < 0:
    c1 = pow(c1, -1, n)
    a = -a
if b < 0:
    c2 = pow(c2, -1, n)
    b = -b

m_squared = (pow(c1, a, n) * pow(c2, b, n)) % n

# m^2 < n であるため標準ライブラリの math.isqrt を用いて整数上で平方根を計算する
m = math.isqrt(m_squared)
print(long_to_bytes(m).decode())
```
